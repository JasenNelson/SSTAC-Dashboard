## 2026-05-10 - Async Button States
**Learning:** Adding explicit visual feedback (like a spinning `Loader2` from `lucide-react`) to asynchronous forms, specifically the Login and Signup auth pages, is an important, high-impact micro-UX improvement. It prevents users from repeatedly clicking the submission button by clearly indicating background processing.
**Action:** When implementing new asynchronous forms or refactoring existing ones across the application, proactively include disabled loading states with clear visual indicators (like spinners).
