## 2025-05-25 - Interactive Color Swatches Accessibility
**Learning:** Visual-only interactive elements like color swatches (`<button>` tags with background color styles) require explicit `aria-label` attributes for screen reader support, as `title` attributes alone are insufficient for accessibility. Furthermore, to support keyboard navigation, they must include clear focus indicators (e.g., `focus-visible:ring-2 focus-visible:ring-sky-500 focus-visible:outline-none`) because default focus styles are often not visible enough when navigating between brightly colored elements.
**Action:** When implementing or reviewing any visually-driven selection buttons (like color pickers or icon-only buttons), always enforce the presence of semantic `aria-label`s and test keyboard focus states.
