## 2026-03-30 - Navigation Menu ARIA Attributes
**Learning:** Next.js Server Components and client-side toggle states require careful ARIA synchronization. Using \`aria-expanded\`, \`aria-haspopup\`, and \`aria-controls\` ensures screen readers announce the state of mobile and desktop dropdown menus accurately. Furthermore, \`aria-hidden="true"\` is critical for decorative icons within interactive elements.
**Action:** Next time, remember to audit interactive components for accurate \`aria-controls\` IDs linking the button and its associated hidden content pane.
